# Отчёт: бенч новостных моделей (game slice, 2026-04-12)

Агрегированные **метрики и выводы** без сырых данных: JSONL со статьями, точками и ответами моделей в репозиторий не входит (`runs/`, часть `datasets/` в `.gitignore`).

## 1. Что измеряли

- **Задача:** по снимку новостей до времени «решения» предсказать сигнал (bias / confidence и т.д., см. схему ответа модели) и сопоставить с **фактической лог-доходностью вперёд** на горизонтах **1d / 3d / 5d** (yfinance, лог-доходности).
- **Метрики (как в `tradenews/benchmark_report.py`):**
  - **Spearman IC** — согласованность **рангов** предсказания и рангов forward return.
  - **95% CI** для Spearman — бутстреп по строкам (одна и та же выборка точек).
  - **p-value (two-sided)** — перестановочный тест для Spearman.
  - **hit_rate_signed** — доля совпадения знака сигнала со знаком доходности.
  - **hit_rate при |bias| ≥ 0.05** — то же при пороге по силе bias (`min_abs_predict=0.05`).
- **Согласованность моделей:** попарный Spearman между **bias** разных моделей на общих строках (`prediction_agreement` в JSON-отчёте).

## 4. Выводы

1. **Короткий и средний горизонт (1d–3d):** **Gemini 2.0 Flash** показывает наибольший Spearman IC и лучшую значимость; **3d** — самый устойчивый результат в этом прогоне.
2. **5d:** **Qwen 2.5 7B** немного выше по точечному IC; нижняя граница CI у Qwen остаётся **выше нуля**, у Gemini — почти у нуля. **GPT** на 5d: CI пересекает ноль — менее надёжно на этой выборке.
3. **Llama 3.2 3B** заметно слабее по ранговой метрике, на **5d** IC отрицательный — для прод-канала «только новости» в текущем промпте/формате приоритет ниже.
4. **Практика на следующие итерации:** опираться на пару **Qwen (локально) + Gemini (облако)**; при необходимости оставить GPT контрольным прогоном. Имеет смысл **скрестить** новостной скор с **техническим** предиктором (отдельная фича/правило согласия), а не заменять технику новостями.
5. **Ограничение:** **43 точки** — один срез; итоги **не** стоит считать универсальной истиной без повторения на другом периоде/универсуме тикеров.
