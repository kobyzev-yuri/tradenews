# Фикстуры tradenews

- `articles/*.json` — массивы новостей в формате nyse `serialize_news_article` (см. `docs/fixtures.md`).
- `minimal_example.json` — минимальный пример для юнит-тестов и ручной отладки предикторов.

Добавляйте свои снимки как отдельные файлы (`mu_2026-04-08.json` и т.д.), не перезаписывая общий минимальный пример без необходимости.
